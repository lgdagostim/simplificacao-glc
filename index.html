Simplificação de GLC

É necessário inserir uma gramática.

Instruções

N Utilize letras em maiúsculo para não terminais.
t Utilize letras em minúsculo para terminais.
-> Utilize para separar os não terminais das produções.
| Utilize para separar as produções.
Σ Utilize como vazio.

Exemplo

J->aABcD|aBcA|bDc
A->bAbD|bb|Σ
B->bDaA|bBBa|bDba|ba|b
D->bB|Σ
